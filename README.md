# My Joke App

A simple React application that fetches and displays random programming jokes using the JokeAPI. Built to practice handling side effects in React with the `useEffect` hook.

## Features

- Display a random programming joke on page load
- Fetch a new joke with a button click
- Show a loading message while waiting for the API response
- Properly handles side effects using React `useEffect`
